# Online-Bank-Mini-System

Objective
Develop and deploy a working banking system with secure transaction validations.

1. Create Account
User must be able to create a new bank account.
2. Deposit Money
The user must be able to deposit money.
3. Withdraw Money
User must be able to withdraw money.
4. Transfer Money
Provide a feature:TransferMoney(senderAccount, receiverAccount, amount)
